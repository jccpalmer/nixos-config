# NixOS configuration

This is a central repo for my NixOS configuration files. I am making active efforts to switch my Linux systems (excluding my Steam Deck OLED and ROG Ally X) to NixOS.

This began with my ThinkPad X1 Carbon Gen 9, but I plan to migrate my Arch-based gaming PC to NixOS after I learn more, modularize my configs, and confirm that I can replicate my system.
